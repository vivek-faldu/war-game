Rules of the game:

1. Each player draws card one by one from the pile. Player with higest value gets both the cards. In case of tie, card
value number of cards are drawn from pile each time and decision is made based on card value of next card.
2. The game can go on forever, so the current version of game declares tie if none of players run out of card in 1000 turns.
3. A player is declared winner if opposite player runs out of card within 1000 chances.
4. Player cannot shuffle cards in middle of game.


Improvements:
1. We can allow players to shuffle cards any time they want.
2. The game can be converted into a web version and can be make interactive with visualization to make it more interesting.
